# Vehicle Price Estimator Project Plan

## Project Goal

Build a full-stack web application that estimates the realistic market
value of a used vehicle.

## Initial Vehicle Inputs

- Make
- Model
- Manufacturing year
- Odometer reading
- Transmission
- Fuel type
- Body type
- Condition
- Australian state or location

## Initial Output

- Estimated vehicle price
- Minimum and maximum price range
- Confidence level
- Explanation of major pricing factors

## MVP Features

1. FastAPI backend
2. Vehicle details input form
3. Baseline price estimation
4. React frontend
5. Price result page
6. Input validation
7. Small vehicle dataset

## Future Features

- Machine-learning price prediction
- Similar vehicle comparisons
- Saved vehicle estimates
- PPSR and registration-related features
- Image-based damage assessment
- Cloud deployment
